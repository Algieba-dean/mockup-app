# Product

## Register

product

## Users
Individual mobile app developers and indie creators who need to generate compliant and beautiful assets for Google Play Store and Apple App Store. They work in fast-paced environments, value speed, privacy, and require a zero-setup, offline-first tool that gets the job done without paywalls or onboarding bloat.

## Product Purpose
To automate and streamline the creation of app store visual assets (framed screenshots, styled banners, multi-version icons) entirely in the browser. Success means a user can drag and drop raw screenshots, customize a multi-page story sequence, and export a complete, compliant, and pixel-perfect ZIP of all required device sizes in under a minute.

## Brand Personality
Clean, Premium, Developer-native. The UI should feel like a high-end developer tool (e.g., Linear, Vercel) rather than a generic design editor.
- **Tone**: Focused, precise, professional, and efficient.
- **3-word personality**: Precise, Efficient, Refined.
- **Emotional Goals**: Inspires confidence, control, and visual satisfaction when exporting.

## Anti-references
- **The Legacy Utility**: Clunky, ad-ridden web tools from 2012 with tiny form controls, slow servers, and disjointed page steps.
- **The SaaS Template Cliché**: Fluffy landing pages with massive cartoon buttons, generic cards with icons, and pushy upgrade alerts.
- **The High-Overhead Canvas Editor**: Bloated design apps (like manual Figma setups) that overwhelm the developer with layout matrices, manual layer management, and complex masking constraints.

## Design Principles
- **Frictionless Velocity**: Actions precede options. Drag and drop screenshots directly onto the dashboard to see immediate, beautiful previews before adjusting any details.
- **Absolute Precision**: What you preview is exactly what is exported. Canvas-driven rendering guarantees pixel-perfect store compliance across all layout modes and aspect ratios.
- **Zero Lock-In & Total Privacy**: No logins, no cloud saving, and no data leaves the browser. Projects are saved locally (via IndexedDB) and run offline-first.
- **Contextual Intelligence**: Sensible, premium defaults are applied instantly. Extract color palettes directly from the user's screenshot to auto-generate matching gradients, typography weights, and shadows.

## Accessibility & Inclusion
- **Developer-Native High Contrast**: High-contrast dark theme by default, ensuring optimal readability under typical developer coding conditions.
- **Keyboard-Friendliness**: Key commands for canvas manipulation, panning, zooming, and navigating text inputs.
- **Motion Restraint**: Option to disable heavy CSS transitions or canvas animations if reduced motion is preferred.
